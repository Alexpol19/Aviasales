# Aviasales - test project bootstraped on create-react-app and MATERIAL-UI  

*To run on your localhost you need node.js v. v12.18.3, npm v. 6.14.6

## How to install
### In the project directory you can run
```bash 
npm install
```
## How to start 
### In the project directory you can run
```bash 
npm start
```
